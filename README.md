# WEAVE
**W**ebsite for **E**xamining **A**utonomous **V**ehicle **E**thics

This website stems from a UMBC Undergraduate Research Award project which is projected to be completed by April 2027. However, this website is projected to be mostly finished (~80%) by September. It will facilitate asking participants what they think about autonomous vehicles (AV) and how they think they should be programmed. 

This project will utilize LLMs in the process of making the website.
